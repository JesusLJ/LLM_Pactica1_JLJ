# Caso práctico 1 - Página web pública

Este proyecto consiste en una landing page pública dedicada a Madrid como destino de viaje. La página se ha desarrollado en un único archivo `index.html`, usando HTML5 semántico, CSS moderno integrado y JavaScript mínimo para mejorar la experiencia de navegación con desplazamiento suave entre secciones.

La estructura incluye un encabezado fijo con menú de navegación, una sección de inicio tipo hero a pantalla completa, un apartado de “Qué ver” con tarjetas de lugares emblemáticos, una sección de transporte con información útil y enlaces oficiales al Metro de Madrid y a la EMT, y un footer profesional. En el pie de página también aparece la referencia solicitada al alumnado, en este caso “Desarrollado por Jesús LJ”, junto con la mención a la actividad para el INAP y su logotipo.

El diseño es responsive y mobile-first, pensado para verse correctamente en móvil, tablet y escritorio. Se han utilizado imágenes públicas de alta calidad para reforzar el aspecto visual y turístico de la web.

Esta práctica responde al objetivo del caso práctico: generar con apoyo de IA una página web pública, temática, atractiva y lista para ser publicada en GitHub y Netlify, entregando finalmente su URL de acceso como evidencia de publicación.
